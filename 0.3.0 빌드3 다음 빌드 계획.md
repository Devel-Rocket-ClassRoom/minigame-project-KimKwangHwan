## 2. 기능 백로그 (우선순위 순)

**P0 — 다음 빌드 필수**

| # | 항목 | 비고 |
| --- | --- | --- |
| 1 | **타이틀 / 메인 메뉴 화면** | 게임 진입점, 새 게임 / 이어하기(세이브 슬롯 선택) 포함 |
