## 3. 잘된 점

1. **맵 시스템 구조화** — `MapManager` + `MapData` SO 조합으로 씬 추가만으로 맵 확장이 가능해졌으며, Additive 로드 방식으로 로딩 부담 분산.
2. **세이브 암호화** — `CryptoHelper`로 세이브 파일을 암호화하여 외부 변조를 차단.
3. **스위치·세이브 연동** — 스위치 활성화 상태를 세이브 데이터에 포함해, 재로드 후에도 문 개방 상태가 유지됨.
4. **계획 초과 달성(맵)** — 목표(1개)를 크게 넘어 9개 맵 전체를 배치하여 다음 빌드의 콘텐츠 기반 마련.

---

## 4. 이슈 / 개선 필요 사항

- **게임 오버 → 세이브 포인트 리스폰** 흐름 미연결 — `GameOverUI` 재시작이 씬 전체를 리로드해 진행 상황 손실 가능
- **타이틀 / 메인 메뉴** 미구현으로 진입점 없음
- **보스 클리어 시퀀스** 미구현 — 엔딩이 없어 게임 한 판이 닫히지 않음
