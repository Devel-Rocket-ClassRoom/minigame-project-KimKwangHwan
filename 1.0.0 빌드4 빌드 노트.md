## 새로 포함된 기능

### 타이틀 / 메뉴 시스템

- **타이틀 씬 완성**: 새 게임 / 이어하기 선택 가능
- **3슬롯 세이브 선택 UI**: 타이틀에서 슬롯별 저장 날짜·상태 표시 후 선택
- **일시정지 메뉴**: ESC 키로 게임 중 토글, `PauseManager.OnPauseChanged` 이벤트 기반
  - 재개(Resume), 타이틀로 돌아가기(Quit to Title) 버튼

### 볼륨 설정 UI (`VolumeSettingsUI`)

- BGM / SFX 볼륨 슬라이더 (범위 0~1)
- `SFXManager`와 직접 연동, 슬라이더 변경 즉시 반영
- `PlayerPrefs`에 자동 저장 → 재실행 후 설정 유지

### 아이템 시스템 (`Item`, `StatModifier`)

5종 스탯 업 아이템 추가:

| 아이템 | 효과 |
|------|------|
| MaxHp 아이템 | 최대 체력 증가 |
| MaxStamina 아이템 | 최대 스태미너 증가 |
| MaxAmmo 아이템 | 최대 탄약 증가 |
| AttackPower 아이템 | 공격력 증가 |
| MaxHealItems 아이템 | 회복 슬롯 증가 |

- 획득 시 `StatModifier`로 즉시 스탯 반영
- 플로팅 텍스트(TextMeshPro): 획득 스탯 정보 표시 후 위로 부상·페이드아웃 (1m / 1s)

### 상자 시스템 (`Chest`)

- 플레이어 접근 시 상호작용 버튼 스프라이트 표시
- 상호작용 입력으로 오픈 → 아이템 스폰 → 0.6s 후 자동 수집
- 오픈 애니메이션 + SFX 재생
- **영속적 상태 저장**: 한 번 열린 상자는 씬 재로드 후에도 열린 상태 유지 (`s_persistentActivated` HashSet → 세이브 데이터 연동)

### 사운드 시스템 (`SFXManager`)

- **BGM 크로스페이드**: 두 개의 AudioSource를 이용한 1.5s 크로스페이드 전환
  - `PlayBGM(clip, crossfade=true)`: 같은 클립 중복 재생 방지
  - `StopBGM(fade=true)`: 페이드아웃 정지
- **SFX 오브젝트 풀**: 16슬롯 라운드로빈 방식, 동일 클립 최대 3개 동시 재생 제한
  - `PlaySFX(clip, volumeScale, pitch)`: 피치 랜덤화 지원
- **볼륨 저장**: `PlayerPrefs("BgmVolume", "SfxVolume")`으로 설정 영속 저장
- **맵 BGM 자동 재생**: 맵 전환 시 `MapManager`가 해당 맵의 BGM 자동 크로스페이드

### 세이브 시스템 확장

게임 시작 시 복원 항목 추가:

| 항목 | 내용 |
|------|------|
| 플레이어 위치 | 저장된 위치 또는 기본 스폰 위치 |
| 플레이어 스탯 | 공격력, 최대HP, 최대스태미너, 최대탄약, 최대회복슬롯 |
| 스위치 상태 | 활성화된 스위치 ID 리스트 복원 |
| 상자 상태 | 열린 상자 ID 리스트 복원 |
| 현재 맵 | 저장된 맵에서 재개 |

### 상호작용 UI 개선

- 모든 상호작용 가능 오브젝트(세이브 포인트, 상자 등)에 버튼 스프라이트 표시
- 상호작용 가능 범위 진입/이탈 시 자동 표시/숨김

### 전투 개선

- **대쉬 공격 슈퍼아머**: 대쉬 중 공격 시 피격 무시
- **ElectricGolem 썬더볼트 애니메이션 수정**: 기존 애니메이션 트리거 동작 안하는 버그 수정

### 카메라 개선

- **초점 전환 기능**: 특정 지점으로 카메라 포커스 이동 후 복귀하는 연출 지원

## 빌드 절차

1. Unity Editor에서 `File > Build Settings` 열기
2. Platform: **PC, Mac & Linux Standalone** 선택, Target Platform: **Windows**, Architecture: **x86_64**
3. **Add Open Scenes** 또는 씬 목록 확인 (TitleScene 인덱스 0 필수)
4. **Build** 클릭 → `Build/` 폴더 선택
5. 출력: `Build/TreasureHunter.exe`

## 알려진 제한사항

- 보스(Witch) 클리어 후 엔딩 시퀀스 미구현 (사망 애니메이션 후 화면 정지)
- ElectricGolem 일부 공격 패턴 미완성
- 게임오버 → 세이브 포인트 리스폰 흐름 추가 검증 필요
- 일부 오브젝트에 SFX 클립 미할당 상태 존재 가능
- 크레딧/엔딩 화면 없음
